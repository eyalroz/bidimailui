Installation

Prerequisites

BiDi Mail UI supports Thunderbird v0.7.x and v0.8.x, as well as Mozilla Suite Mail&News v1.4x to v1.8.x. It has been successfully tested under Windows, Mac OS X and Linux+X.Org. It is most likely, though we can't guarantee it, to work fine with other versions and operating systems.

Installation under Thunderbird

  1. Right click the download link below
  2. Save the XPI file to your hard drive
  3. Launch Thunderbird
  4. Open the Tools menu and choose Extensions
  5. Click Install
  6. Browse to the folder where you've saved the XPI file, select it, then confirm
  7. Restart Thunderbird when prompted
The Thunderbird Extension

Installation under Mozilla

  1. If previous BiDi Mail UI versions are installed, make sure to remove them as described in section 3
  2. Click the download link below
  3. Confirm the installation when prompted
  4. Choose whether you wish to install the extension in your personal profile folder or globally, in the application folder. Installation in the profile folder has a few advantages: There is no need to have root user privileges (in operating systems such as Linux) and there is no need to reinstall the extension after updating Mozilla to a new version. However, with operating systems that are oriented toward single user mode (such as Windows 98), but that are accessed by several users, it is preferable to install BiDi Mail UI in the Mozilla application folder, so that everyone can utilize it. This global installation may require root privileges in some operating systems. Note that installing the extension as root also requires to restart Mozilla as root.
  5. Restart Mozilla when prompted. Make sure to close all the program's windows - Browser, Mail&News, Composer and so forth.

The Mozilla Extension

To overcome possible difficulties in step #3, it may be worthwhile to run the installation locally. Simply right click the XPI file, save it to your hard drive and double click it to start the installation.

Removal

If you wish to uninstall BiDi Mail UI, please follow the instructions that correspond to your chosen installation method. Further to that, if for some reason, your experience with the extension was less than satisfactory, please let us know. You can find our contact details above.

Removal of BiDi Mail UI from Thunderbird

Open the Extensions manager (Tools -> Extensions). Locate the BiDi UI entry and click Uninstall.

Manual removal of BiDi Mail UI from Mozilla 1.x Suite application folder

The following instructions only apply to Mozilla, not to Thunderbird.

  1. Open Mozilla's chrome folder
  2. Edit the file installed-chrome.txt and delete any line that contains the string "bidimailpack.jar".
  3. Delete the file bidimailpack.jar
  4. Delete the folder overlayinfo and any file contained within it
  5. Delete the file chrome.rdf
    6. Restart Mozilla (in Linux, do it as root if needed). Mozilla will recreate the overlayinfo folder and the chrome.rdf file that were deleted in steps #4 and #5.

Manual removal of BiDi Mail UI from Mozilla 1.x Suite user's profile folder

The following instructions only apply to Mozilla, not to Thunderbird.

  1. Open Mozilla's chrome folder
  2. Delete the file bidimailpack.jar
  3. Edit any of the overlay.rdf files that appear under the overlayinfo sub-directory. Delete the tags beginning with RDF:Seq RDF:about that include lines with reference to bidimailpack (these may be multi-line).
  4. Edit the file chrome.rdf under the chrome folder and delete the lines that begin with RDF:li and contain references to bidimailpack, as well as tags that begin with RDF:Description RDF:about and contain references to bidimailpack (these too may be multi-line).
  5. Restart Mozilla
