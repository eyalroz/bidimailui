About BiDi Mail UI

Users of languages that are written from right to left, such as Arabic, Persian and Hebrew, require mail client software that is capable of displaying mail messages using the intended direction. This special functionality is especially important in the case of plain text messages, as those do not explicitly specify direction. Similar direction control is required when composing new messages.

The BiDi UI extension provide access to the inherent Bi-Directional capabilities in Mozilla Mail clients (Mail&News component in Mozilla 1.x Suite and the separate application Thunderbird) - both for display and composition.

Background

BiDi UI is the new incarnation of another extension that bore the same name. The previous extension was targeted for Mozilla browsers and was created to solve Bug 85420 . Note that this bug has been fixed in the meanwhile, but it is hasn't been integrated yet to an official release of Mozilla browsers (this is expected to change soon). Another related extension that preceded this one is Moofie's HebMailPack which was developed in order to fix Bug 119857 ; The new BiDi UI extension is a collection of modifications and improvements based on Moofie's original code.
