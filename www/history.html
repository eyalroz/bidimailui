History
0.1
  • First public release.
0.2
  • UI for controlling page direction has been added to the View menu.
  • "Switch Text Direction" instead of "Invert Direction".
  • In addition to the contextual menu, "Switch Text Direction" command is now available in the Edit menu.
0.2.5
  • In addition to the View menu item, "Switch Page Direction" is also available in the page contextual menu.
  • Both commands have been moved to the end of the contextual menu
  • In the View menu, "Switch Page Direction" command has been moved after "Character Encodings"
0.3
  • Localization for Hebrew has been added.
0.3.1
  • Accel+9 has been defined as the keyboard shortcut for "Switch Text Direction".
  • Mnemonics are also available for both commands.
0.3.2
  • Fixed an issue where "Switch Text Direction" didn't work in some cases.
  • Mozilla Suite version.
0.3.3
  • Changed the keyboard shortcut to Accel+Shift+D - across platforms (Firefox Only).
  • Consolidate both extensions into a single package.
0.3.4
  • Accel+Shift+E is the keyboard shortcut for "Switch Text Direction" in both Firefox & Mozilla Suite.
